# FRIZZY

## Aplikacija link - https://frizzy-woad.vercel.app/

* Razvio sam kompletnu Full-stack web aplikaciju za automatizaciju rezervacija u frizerskim salonima 

### Ključne funkcionalnosti:
* Role-Based Access Control: Implementirao sam tri razine pristupa (Admin, Zaposlenik, Klijent) s personaliziranim sučeljima i sigurnosnim rutama.

* Dinamički sustav rezervacija: Izradio sam interaktivni kalendar s logikom za provjeru dostupnosti termina u stvarnom vremenu (slot management)
i filtriranjem po frizerima/uslugama.

* Moderni UI/UX: Dizajnirao sam responzivno sučelje koristeći najnoviji Tailwind v4.

* Nadzorna ploča (Dashboard): Za admin korisnike razvio sam sustav za upravljanje resursima (zaposlenici, cijene, usluge),
dok zaposlenici imaju uvid u dnevni raspored s brzim pristupom kontakt podacima klijenata.

* State Management & API: Upravljanje stanjem aplikacije (autentifikacija, tokeni, podaci o terminima) 
kroz React hookove i asinkronu komunikaciju s backendom.

### Tehnološki Stog (Tech Stack):
* Frontend:   React, Vue.js, JavaScript, Tailwind CSS
* Backend:   Node.js, Express.js
* Baza podataka & ORM:   PostgreSQL (Supabase), Prisma ORM
* Sigurnost:   JWT autorizacija, bcrypt enkripcija lozinki
* Testiranje:   Playwright (E2E testiranje), Postman (API testiranje)
* Deployment:   Vercel (Frontend), Render.com (Backend)
